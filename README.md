# Leitostrap
## FFlags After Patch
### Developed by: Leito and Prozone.
**thanks for helping me other developers**

[+] FFlag Injector
Advanced FFlag injection engine utilizing memory pattern scanning to ensure compatibility without external offset files.

[+] Anti-Debug & Anti-VM
Multi-layered protection system featuring IsDebuggerPresent, NtQueryInformationProcess, and environment detection for VMware/VirtualBox.

[+] Memory Pattern Scanner
Dynamic localization of memory offsets to eliminate the need for manual updates or external dependencies.

[+] Process Manager
Deep client integration with the ability to suspend and resume threads via NtSuspendProcess for stable injection.

[+] Integrity Guard
Security module that verifies executable integrity using SHA-256 hashing to prevent unauthorized tampering.

[+] Mods (Coming soon!)
Local asset replacement system for character bundles (Korblox, Headless), custom skyboxes, and environmental textures.

[+] Custom Cursors & Themes (with bugs)
Fully customizable visual experience including a variety of custom cursors and high-contrast themes (Dark, Glass, Animated, Etc...). 

[+] Web-UI Integration
Modern hybrid interface powered by pywebview, featuring smooth transitions and dark mode support.

[+] Community API
Real-time configuration syncing with external databases (Supabase) to download and share community-made presets.

[+] Discord RPC
Rich Presence integration via pypresence to display your current status and software version on your profile.

[+] Global Hotkeys
Instant UI toggle and visibility control using global keyboard hooks (default: Delete key).

[+] Bug Fixes & Optimization
Continuous refinement of internal logic to resolve common FFlag conflicts and improve overall execution stability.
